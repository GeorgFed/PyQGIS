## Задание
Написать алгоритм для отображения рельефа на местности. 

Алгоритм: 
qgis:relief

Данные:
Выбрать данные для обработки необходимо самостоятельно. 
Сделать это можно на сайте:
https://srtm.csi.cgiar.org/srtmdata/

Результат:
Необходимо получить рельефную DEM в виде .tif файла в произвольной папке. 

Дополнительное задание:
Самостоятельно выбрать цвета для отоборажения рельефа
